# Evil Hangman
Similar to the traditional Hangman game, in Evil Hangman instead of picking a single word, the computer picks a set of words that fit the current pattern. Beat the computer by guessing the correct word to win the game!

## Technologies
* Java v24.02
* java swing

## Installation and Setup
``` 
git clone https://github.com/IGrewal08/Hangman-Game-Java
cd Evil Hangman
```

### Run this project
/src/Main.java entry point
